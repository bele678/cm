pycharm 快捷键

指法是

1.左手小拇指按下键盘左下角 Ctrl 键，不放手。

2.左手其他手指按下另外一个按键一次然后松开。

3.左手小拇指松开 Ctrl键。

4.快捷注释时，一般用右手，同理操作主键盘区的右下Ctrl键。

保存 Ctrl + S

复制 Ctrl + C

粘贴 Ctrl + V

剪切 Ctrl + X

撤销一步修改 Ctrl + Z

快捷注释 Ctrl + /

取消快捷注释 Ctrl + /
