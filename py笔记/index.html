5.代码块

代码块是有冒号所在的一行与下面有4个缩进的代码共同组成的。

# if语句 伪代码（.表示空格，实际编写代码是写空格）

if 条件:

....代码0

....代码1

else:

....代码2

....代码3

# 代码含义： 如果条件成立，则执行代码0，代码1。如果条件不成立，则执行代码2，代码3.

6.and or 逻辑运算

# and 语句伪代码

if 条件1 and 条件2:

....代码0

....代码1

else:

....代码2

....代码3

#代码含义：所有条件全部成立时，才执行代码0，代码1。有任意1个不成立就不执行代码0，代码1，他会执行代码2，代码3

# 如果是用or,那么任意1个成立就算整个条件成立，执行代码0，代码1部分代码。

7.字符串截取（取字符串中间的文本）

# 字符串截取中间

str[x] #取字符串str的索引x的值

str[x2:x3] #取字符串str的索引x2到索引x3之间的值，包含x2索引的，不包含x3索引的。

str[x2:] #取字符串str的索引x2开始到后边的全部值，含x2索引的，也含结尾的。

str[:x3] #取字符串str的开头到索引x3之间的值，含开头，不含x3索引的。

如: str ="abc1234" 那么， 取里面元素时，可以认为 str = ["a","b","c","1","2","3","4"] 可以使用list 列表的方法进行操作里面的元素/值.

2.文件全名

文件全名 有 盘符+路径+主文件名+扩展名 组成

例子：
D:\其他文件\机械硬盘\desktop\file.py
d: 是盘符
\其他文件\机械硬盘\desktop\ 是路径，位置
file.py 主文件名+扩展名
